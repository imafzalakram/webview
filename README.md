## Features

 - 基本配置使用(宽度自适应、返回网页上一层、显示网页标题等)
 - 唤起三方应用(拨打电话、发送短信、发送邮件等)
 - 上传图片(版本兼容)
 - 错误页面处理
 - 全屏播放网络视频
 - **与Js交互实例**
 - 优雅的进度条显示控件
